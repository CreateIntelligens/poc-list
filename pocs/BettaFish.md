# POC: BettaFish 輿情洞察系統

## 概述
一套 AI Agent 協作的輿情監測與分析系統，能自動巡檢 30+ 國內外主流社群平台與媒體站點，整合文字、圖片、影音資料，輸出結構化決策級報告，幫助企業及時掌握公眾輿論變化。

## 使用方法
1. 訪問(內網) http://localhost:8903/
2. 輸入監測關鍵字或品牌名稱
3. 設定監測時間範圍和目標平台
4. 點擊「開始分析」，系統自動啟動多引擎並行處理
5. 實時查看分析進度和初步觀測
6. 等待完整報告生成（包括情感分析、趨勢洞察、建議方案）
7. 下載或導出 HTML 報告及相關媒體資產

**使用建議**：
- 關鍵字越具體分析效果越好
- 建議監測時間跨度 7-30 天以獲得完整趨勢
- 支援多平台同時監測，可自訂平台組合

## 主要功能
- **多平台監測**：支援 30+ 國內外主流社群與媒體平台
- **四大引擎協作**：Query、Media、Insight、Report 引擎同步運作
- **情感分析**：多種情感模型集合，自動語言判斷
- **多模態理解**：整合文字、圖片、影音資料
- **結構化數據提取**：自動萃取貼文、評論、轉發等結構化資訊
- **決策級報告**：生成 HTML 報告含趨勢圖表與建議方案
- **策略調整**：透過 Forum Agent 主持，動態優化搜尋策略

## 應用場景
- 品牌輿論監測和危機預警
- 競品分析和市場動態追蹤
- 產品上市前市場反應評估
- 新聞事件和熱點話題監控
- 客戶情緒和滿意度監測
- 公關活動效果評估
